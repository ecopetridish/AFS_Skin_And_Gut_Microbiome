**Antarctic fur seal (Arctocephalus gazella) skin and gut microbiome project**

In this project we used 16S rRNA amplicon sequencing to describe the skin and gut Antarctic fur seal microbiome in relation to host-related (e.g. age class, sex) and environmental factors (breeding colony). 

In this repository you can find the i) data and ii) scripts that were used for the data analysis, as well as the iii) figures that were produced (folder "out").
